# Script Engine (Writer)

Serverless-shaped. Generates and refines scripts via a single continuous chat session (rough idea -> polish -> structure -> generate -> refine), backed by a LangChain generation chain plus a stateful multi-turn chat refinement layer with prompt caching. See `../../AI_Studio_PRD.md` Section 4 and Naming Conventions.

No feature code yet - foundation scaffold only, Week 5 Day 1.

## Local dev

From repo root: `uv sync`, then `uv run --package script-engine <command>`.
